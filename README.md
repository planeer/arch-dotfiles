# arch-dotfiles
Dotfiles for arch linux 

![Screenshot](https://raw.githubusercontent.com/planeer/arch-dotfiles/master/screenshots/screenshot.jpg)

### Symlinks
* ~/ & /root/:
    * .zshrc
    * .zprofile
    * .gtkrc-2.0
    * .compton.conf
    * .vimrc
* ~/.config/:
    * i3
    * alacritty
    * ranger
    * rofi
    * polybar
    * gtk-3.0
    * dunst
* ~/.bin/scripts/:
    * lock/lock.sh
* ~/Documents/:
    * lock/lock.png
