# arch-dotfiles
Dotfiles for arch linux 

![Screenshot](https://raw.githubusercontent.com/planeer/arch-dotfiles/master/screenshots/screenshot.jpg)

### Symlinks
* ~/ & /root/:
    * .zshrc
    * .zprofile
* ~/.config/:
    * i3
    * termite
    * ranger
    * rofi
    * polybar
* /etc/:
    * i3status.conf
* ~/.bin/scripts/:
    * lock/lock.sh
* ~/Documents/:
    * lock/lock.png
